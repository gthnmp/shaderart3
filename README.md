# Three.js Shader Art 

A Three.js shader art experiment

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)

## Installation
```bash
# Clone the repository
git clone https://github.com/gthnmp/threejs-shader-art3.git

# Change directory
cd threejs-shader-art3

# Install dependencies
npm install
```
## Usage
```bash
# run project 
npm run dev
```
